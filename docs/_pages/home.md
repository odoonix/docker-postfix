# Odoo Mail Gateway Documentation

Build and operate a Postfix + FastAPI email gateway designed for Odoo-centric workflows.

[Get Started](/resources/) | [Read Blogs](/blog-list/) | [Open Tutorials](/tutorials/)

## What This Project Provides

- Postfix-based inbound and outbound mail gateway
- API-driven management for mail routing and policies
- Odoo-compatible architecture for multi-tenant operations
- Docker-ready deployment for reproducible environments

- **Resources**: setup references, technology links, and project entry points
- **Blogs**: announcements, roadmap updates, and architecture notes
- **Tutorials**: practical development and run guides from `docs/_dev`
